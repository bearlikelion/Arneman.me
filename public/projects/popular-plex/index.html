Popular Plex

Oct 27, 2020 · 50 words · 1 minute read python plex tatulli source code project

This script is designed to run weekly, generating collections in Plex of most popular Movies and Television based on the data from the Tautulli API. It uses the python-plexapi package and requests.
The idea behind this project was to recreate Netflix’s ‘What’s Popular’ list
